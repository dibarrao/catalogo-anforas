📁 Instrucciones: Uso con OneDrive / SharePoint

Para agregar imágenes nuevas:

  1. Renombra la imagen con el código exacto del producto (ej: ANF0015.jpg)
  2. Súbela a la carpeta imagenes/ en OneDrive, dentro del mismo directorio que el HTML
  3. El catálogo la mostrará automáticamente en todos los equipos sin pasos adicionales
  4. ✅ No necesitas hacer nada más — funciona para todos los usuarios

Para actualizar stock:

  1. Exporta el nuevo stock desde tu sistema ERP/Navision a Excel
  2. Guárdalo como stock.xlsx
  3. Accede al Panel de Administrador en el catálogo
  4. Usa "Importar desde Excel" para actualizar todos los stocks
  5. El catálogo refrescará los datos automáticamente

💡 Estructura de archivos requerida: Sube a OneDrive/SharePoint la carpeta CATALOGO/ que contiene: catalogo_stock.html + subcarpeta imagenes/ con todas las fotos. Las imágenes aparecerán automáticamente en todos los equipos. Solo tú puedes modificar stock o agregar imágenes con el PIN 8503.
